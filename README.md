# In-Cub
Florentin et Guillaume

## Installation 2 façons

### 1 ère façon
- fonctionne unniquement sous windows
```
- start install.bat
```

### 2 ème façon
```
- cd tp_in-cub
- npm install
- cd tp_in_cup_api/in-cub-api
- npm install
```
## Lancement

### Le serveur 
```
- cd tp_in_cup_api/in-cub-api
- node .
- http://localhost:3000
- http://localhost:3000/explorer
```

### le client
```
- cd tp_in_cup/src
- ng serve
- http://localhost:4200
```
